# kanirobocon-system / カニロボコン運営システム

カニロボコン得点システム、クライアント側 for iPad / PC

## 起動方法

[Deno](https://deno.land/)をインストールし下記で実行
```bash
deno run -A kanirobo-server
```
- [http://localhost:8000/](http://localhost:8000/)を開く
